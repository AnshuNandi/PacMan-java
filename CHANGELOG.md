# Changelog

## v1.0.0 (2025-12-24)

Initial public release.

Highlights
- Classic Pac-Man style gameplay built in Java Swing
- Ghost AI with scatter/chase modes and distinct personalities
- Power pellets, frightened ghosts, cherry fruits
- Level progression, score + high score tracking, lives
- Background music and procedural sound effects
- Packaging: Executable JAR and Windows EXE installer (bundled JRE via jpackage)
